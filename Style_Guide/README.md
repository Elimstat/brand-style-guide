# Elimstat Brand Style Guide

Official brand guidelines and visual standards for Elimstat.com.

## Contents

This repository contains:
- **Elimstat_Brand_Style_Guide.pdf** - Comprehensive brand style guide with color palettes, typography, logo usage, and design standards
- **elimstat-brand-guidelines.skill** - Claude Code skill for applying Elimstat brand styling to artifacts

## Brand Resources

The brand style guide includes:
- Color palette specifications
- Typography standards
- Logo usage guidelines
- Design patterns and components
- Brand voice and messaging guidelines

## Using the Brand Guidelines

Designers, developers, and content creators should reference the brand style guide to ensure consistency across all Elimstat properties and materials.

### Claude Code Skill

The `elimstat-brand-guidelines.skill` can be used in Claude Code to automatically apply Elimstat brand colors and typography to generated artifacts.
