DOCUMENT OVERVIEW

This style guide defines the typography, colors, layout, and formatting standards for all Elimstat product specification sheets. These specifications ensure brand consistency and professional presentation across all ESD monitoring equipment documentation.

PRIMARY BRAND COLOR
Pantone 2748 UP (Elimstat Blue)
RGB: 10, 48, 140 | Hex: #0A308C | CMYK: 100, 83, 0, 32

This deep, professional blue conveys trust, technical precision, and authority—perfect for industrial ESD equipment documentation. Use for all graphic elements, headers, and brand accents.

STANDARD DOCUMENT STRUCTURE

All Elimstat spec sheets follow this consistent section order:

  1. Document Title — Product name and series identifier
  2. Product Description — Overview paragraph with key features
  3. Physical Properties — Material, dimensions, weight, color options
  4. Electrical Properties — Resistance specifications, static decay rates
  5. Specifications Table — Detailed technical data using Bennett Table Style
  6. Standards & Compliance — ANSI/ESD references, RoHS, Lead-Free certification
  7. Testing Procedures — General test procedures (when applicable)
  8. Material Handling — Application instructions (when applicable)

COMPLIANCE STANDARDS

Standard Compliance Language:

"This product meets the ANSI/ESD S20.20 STM S4.1 Standard using FTMS 101C Method 4046. Material is Lead-Free. RoHS Compliant."

All specification sheets must reference these industry standards:

  • ANSI/ESD S20.20 — Electrostatic Discharge Control Program Standard
  • STM S4.1 — Worksurfaces Resistance Measurements
  • FTMS 101C Method 4046 — Federal Test Method Standard
  • RoHS Compliance — Restriction of Hazardous Substances Directive
  • Lead-Free Certification — Federal compliance for lead content
LAYOUT & SPACING

Keyboard Shortcuts (InDesign):

Option + Command + 1 — Hair space (fine-tuning)
Option + Command + 6 — En space (tables and aligned text)
~ — Sixth space (bullet point spacing)

Leading Ratios: The style guide uses intentional leading (line spacing) to enhance readability:

  • 12pt text → 18pt leading = 150% ratio (excellent for body text)
  • 13.2pt text → 15.84pt leading = 120% ratio (tighter for headers)
  • 10pt text → 12pt leading = 120% ratio (compact for footnotes)
BRAND CONSISTENCY CHECKLIST

Before finalizing any spec sheet, verify:

  • Helvetica Bold 18pt used for document title
  • Museo Sans used for all body text with correct weights
  • Pantone 2748 UP blue applied to graphic elements
  • Blue Graphic Line 2 present in header
  • Blue Line Footer present at bottom
  • Elimstat logo included at correct scale
  • Bullet points use sixth space (~) spacing
  • Tables use Bennett Table Style
  • All compliance statements included
  • Tolerance specifications included (where applicable)
  • ANSI/ESD standards referenced correctly
  • RoHS compliance statement present
